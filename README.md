# applied_stats_report
This repository contains an applied statistics assignment analysing the relationship between air pollution and respiratory difficulties.  
The analysis is conducted in R and includes exploratory data analysis, logistic regression modelling, diagnostics, and prediction.  
The full methodology and results are documented in the accompanying PDF report.  
The original dataset provided with the assignment is included for transparency and reproducibility.  
The assignment questions are included to provide context for the analysis and modelling choices.
